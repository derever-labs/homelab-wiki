# Querschnitt

Dieser Bereich enthaelt Anleitungen und Konzepte, die mehrere Systeme gleichzeitig betreffen. Waehrend die Service-Seiten jeweils ein einzelnes System dokumentieren, beschreiben die Querschnitt-Seiten das Zusammenspiel und uebergreifende Architekturentscheidungen.
